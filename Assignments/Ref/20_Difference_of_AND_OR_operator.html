20. What is the difference between && and || in JavaScript?

The logical AND (&&) operator for a set of boolean operands will be true if and only if all the operands are true. Otherwise it will be false.

The logical OR (||) operator for a set of operands is true if and only if one or more of its operands is true.

it returns a Boolean value, so if this operator is used with non-Boolean values, it will return a non-Boolean value.

 <script>
 const a = 3;
 const b = -2;

 console.logspan (a > 0 && b > 0);  // Expected output: false

 console.logspan (a > 0 || b > 0);  // Expected output: true
 </script>
